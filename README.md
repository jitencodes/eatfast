### Run Locally

To run **eatfast** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/jitencodes/eatfast.git
```

Windows:

```bash
git clone https://github.com/jitencodes/eatfast.git
```
